Computer Architecture Lab Repository
This repository contains a comprehensive collection of Computer Architecture laboratory experiments, including Lab 1, Lab 2, and additional labs. Each lab folder contains detailed documentation and implementation files such as README files, simulation codes, testbenches, output results, and related resources.
Every lab README includes:
Title
Objectives
Theory
Discussion
Output/Results
Conclusion

The repository serves as a structured reference for understanding and implementing fundamental Computer Architecture concepts through practical experiments.
